# About our Project (eNutri)
eNutri is a complete ecosystem that consists of: 
- Nutri Dispenser
- Nutri App
- Nutri Web.

# Nutri Dispenser
## The Hardware
Nutri Dispenser is 3D Printed and built with:
- Raspberry Pi 4
- 16-bit LCD Screen
- Stepper Motor
- Linear Actuator
- H-Bridge
- Relay
- DC Water Pump
